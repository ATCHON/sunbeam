============================================
Sunbeam: an iridescent metagenomics pipeline
============================================

Sunbeam is a software pipeline written in `snakemake <http://snakemake.readthedocs.io>`_
that simplifies and automates many of the steps in metagenomic sequencing
analysis.

To get started, see our :ref:`quickstart`!

Installation
************

To install, clone or download the repository and run

.. code-block:: bash

   bash install.sh


Usage
*****

Activate the Sunbeam environment with ``source activate sunbeam``; deactivate
using ``source deactivate``.
